# AI Resume Analyzer UI Design

This project is divided into two main parts:

1. **Frontend**: Contains the user interface built with React.
2. **Backend**: Contains the server-side logic built with FastAPI/Express.js.

## Setup Instructions

### Frontend
1. Navigate to the `frontend/` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend
1. Navigate to the `backend/` directory.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the backend server:
   ```bash
   uvicorn main:app --reload
   ```

### Environment Configuration
- Add `.env` files in both `frontend/` and `backend/` directories for environment-specific variables.

### Running the Application
1. Start both the frontend and backend servers.
2. Open the frontend in your browser and interact with the application.
